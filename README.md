# Praxintesis-Landing-Page

Landing page estática separada en `index.html`, `styles.css` y `script.js`.

## GitHub Pages

1. Sube estos archivos a la rama que usarás para Pages.
2. En GitHub, ve a `Settings` > `Pages`.
3. En `Build and deployment`, elige `Deploy from a branch`.
4. Selecciona la rama y la carpeta raíz `/ (root)`.
5. Guarda los cambios y espera a que se publique el sitio.

## Desarrollo local

Abre `index.html` en el navegador o con una extensión como Live Server.
